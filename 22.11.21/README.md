### 명령어
man(메뉴얼)

    메뉴얼을 보여줌
   

* 단어 지우기 : ctrl+W
* 행 지우기 : ctrl+u
* 글씨 키우기 : ctrl+ +
* 글씨 줄이기 : ctrl + -

echo 
    
    기능 : 화면에 한 줄의 문자열 출력
    형식 : echo [-n] [문자열]
    옵션 : -n :마지막에 줄 바꿈을 하지 않음.


printf 

    기능 : 자료를 형식화하여 화면에 출력
    형식 : printf [형식] [인수]

셸 기본 사용법 

변수 정의

    형식 : 변수명=문자열
    예시 : SOME=test
    

alias

    기능 : 별칭 생성
    형식 : alias 이름='명령'
    예시 : alias l= 'ls -CF'
    	   alias la= 'ls -A'
    
history
	
    명령 입력 기록을 출력해준다.
